# Alura Play - Proyecto Flexbox y Layouts Responsivos

Este proyecto es una implementación de la plataforma Alura Play, diseñada para mostrar videos de forma atractiva y responsiva. Se utilizó Flexbox para crear un diseño adaptable que se ajusta a diferentes tamaños de pantalla.
## Tecnologías

* HTML
* CSS

## Instrucciones de uso

1. Clona este repositorio: `https://github.com/edurivas23/Aluraplay.git`
2. Abre el archivo `index.html` en tu navegador web.

## Funcionalidades

* **Diseño responsivo:** El diseño se adapta a diferentes tamaños de pantalla, utilizando Flexbox para un diseño flexible.
* **Encabezado:** El encabezado contiene el logo de Alura Play y un menú de navegación.
* **Menú lateral:** El menú lateral muestra las categorías de videos.
* **Sección de videos:** La sección de videos muestra una lista de videos con información relevante.
* **Pie de página:** El pie de página contiene información de contacto y derechos de autor.
